Class

OOP (Object Oriented Programming style) murni digunakan dalam contoh ini meskipun browser masing masih akan mengkompilasinya menjadi model prototype. Semoga tak lama lagi, penerapan OOP (typed script) dimanfaatkan browser untuk meningkatkan kinerja (performance) karena lebih mudah dikompilasi ke bahasa asli (native).

Javascript, mulai sejak standar ECMA 2015 (ES-6) memperkenalkan class sehingga kita dapat menyajikan penulisan program (scripting) obyek yang mudah dipahami.

Class adalah suatu obyek yang memiliki property (variables) dan metode (method / function). Keberadaan class membuat kita dapat menerapkan gaya OOP dengan sempurna. Class dapat dikembangkan (extends) menjadi kelas-kelas turunan (inheritance) lain, seperti kelas binatang menjadi kuda, lalu menjadi kuda kecil dan kuda besar. Masing-masing memiliki karakteristik, prilaku dan kemampuan yang spesifik.

Contoh Class

Berikut contoh class yang kita gunakan:
