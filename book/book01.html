Seri Memahami 3D Engine

Engine 3D

Engine 3D merupakan perangkat lunak (sofware) untuk menampilkan model 3D ke layar. Jika anda browsing di di dunia maya, akan banyak sekali anda temui contoh-contohnya. Sayangnya, umumnya demikian kompleks sehingga sulit bagi pemula untuk dapat memahami cara kerjanya secara cepat. Untuk memudahkan anda dalam memahaminya, saya sajikan contoh Engine 3D sederhana dan dengan menggunakan javascript sehingga perhatian anda tidak terpecah pada kompilasi dan SDK (development kit). Javascript tingga anda panggil dari html dan browser mengkompilasinya bila perlu.

Efek 3D

Ilusi 3 Dimensi yang disajikan di layar pada umumnya memanfaatkan distorsi image (gambar) yang dapat dioptimalisasi dengan segitiga (triangle) untuk menghasilkan ilusi 3 dimensi.

Triangle memberi kita kemudahan untuk memperoleh tiga titik (point) yang dapat dijadikan dasar untuk memiringkan (shearing). Jarak atau posisi z triangle memberi kita informasi untuk menghitung skala.

Berikut menrupakan contoh bagaimana hal tersebut dapat terjadi.

Topik Bahasan

Kita akan membuat satu Engine 3D yang cukup lengkap untuk menampilkan obyek 3D akan tetapi tetap sederhana sehingga mudah dipahami. Pembahasan akan dilakukan bertahap, meskipun anda dapat lompat sesuka anda, dengan topik utama sebagai berikut:

  1. Transformasi dan Proyeksi. Berkaitan dengan perubahan perhitungan posisi relatif antara titik dengan induknya (node) yang bertingkat-tingkat hingga ke kamera.
  2. rendering, yaitu penempatan hasil proyeksi ke kordinat layar, termasuk penggunaan gambar sebagai tekstur.

Selanjutnya silahkan menelusuri menu yang ada di samping.
