MATERIAL

Material merupakan obyek (class) yang digunakan untuk menyimpan data warna atau tekstur dan melakukan proses terkait sesuai keperluan.

Kali ini kita gunakan metode paling sederhana sehingga skripnya cukup sebagai berikut:

Skrip Material

                    

class ColorMaterial {
    constructor(bg, bordercolor, borderwidth) {
        this.backgroundColor = bg;
        this.borderColor = bordercolor;
        this.borderWidth = borderwidth;
    }
}

class TextureMaterial {
    constructor(img) {
        this.image = img;
    }
}

ColorMaterial

  • backgroundColor, untuk warna latar.
  • borderColor, untuk warna garis tepian.
  • borderWidth, untuk ketebalan garis tepian.

TextureMaterial

Hanya memuat data image, karena di CanvasRendering2d kita hanya butuh itu.

Lebih menyenangkan lagi, kita tidak perlu membuat Getters and Setters maupun methods.

Sedikit Mengenai Class

Class adalah obyek yang kita bentuk untuk menyimpan data dan melaukan sesuatu dengan keunikannya, seperti individu organik. Class dapat pula dikembangkan turunannya sebagaimana di biologi ada sepecies, ordo dan seterusnya.

Di atas, merupakan contoh class yang hanya memiliki konstruktor. Secara lengkap, kelas dapat memiliki:

  • Property, yaitu data yang disimpan di kelas tersebut.
  • Getters dan Setters, yang sering dimasukkan sebagai bagian dari property karena merupakan cara mengubah property yang penggunaannya sama dengan cara biasa mengubah property.
  • Methods, yaitu fungsi atau prosedur yang dapat dilakukan oleh kelas tersebut.

Berikut contoh class sederhana tapi memiliki unsur yang lengkap:



class Organic {

    constructor (group_name) {

        this.group = group_name;

    }
}

// Kita dapat membiat turunannya dengan deklarasi extends, misalnya:

class Animal extends Organic {

    constructor(species_name) {
        /*
        * Memanggil constructor induk sehingga nama group "animal" tersimpan
        * dan semua property maupun method di induk menjadi bagian anak, 
        * kecuali jika diubah kemudian:
        */
        super ("animal");
        // Tambahan property baru:

        this.species = species_name;
        this._words   = ["I am", "my species is"];

    }
    // Getter and Setter:
    get firstWord() {
        return this._words[0];
    }
    set firstWord (text) {
        this._words[0] = text;
    }
    get secondWord() {
        return this._words[1];
    }
    set secondWord (text) {
        this._words[1] = text;
    }
    // Method:
    whatAreYou () {
        return this._words[0] + " " + this.group + " " + this._words[1] + " " + this.species;
    }
}

// sehingga kita dapat membuat obyek baru dan memanfaatkannya sebagai berikut: 

window.addEventListener("load", (e) => {

    let dog = new Animal("DOG");
    dog.firstWord = "Saya termasuk";

    let s1 = dog.firstWord;
    let s2 = dog.secondWord;
    let s3 = dog.whatAreYou();

    let s = "First word : " + s1;
    s += "\nSecond word : " + s2;
    s += "\n\nDo'i ngomong : '" + s3 + "'";
    
    alert (s);
});
